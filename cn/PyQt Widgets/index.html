1 简介与安装

1.1 Python 图形界面开发的几种选择

  1. Tkinter: 基于 Tk，稳定，但控件少，适用于小型应用开发；
  2. wxPython: 基于 wxWidgets，文档与用户少，不适用于大型应用开发；
  3. PySide2/PySide6/PyQt5/PyQt6: 基于 Qt，控件丰富，文档与用户多，适用于大型应用开发。

1.2 在PyCharm 中配置 PyQt5

  1. 在 Python 环境中安装 PyQt5：
  1. 文件 > 设置 > 外部工具中配置与使用 QtDesigner、PyUIC、Pyrcc：

  • QtDesigner

  • PyUIC

  • Pyrcc

.ui 文件转 .py 文件：

  • 点击 EXternal Tools 里面的 QTDesigner，跳转到QT界面，保存为 .ui 文件；
  • 选中文件，鼠标右击，点击 EXternal Tools 里面的 PyUIC，生成 .py 文件。
